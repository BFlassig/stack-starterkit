# Code of Conduct

## Our Commitment

We maintain an open, respectful, and safe collaboration environment for all contributors,
regardless of experience, background, identity, or perspective.

## Expected Behavior

- communicate constructively and respectfully
- take different viewpoints seriously
- frame criticism in a concrete, solution-oriented way
- take responsibility for your own mistakes
- document feedback clearly

## Unacceptable Behavior

- personal attacks, insults, or degrading language
- harassment, discrimination, or intimidation
- publishing private information without consent
- repeated destructive or disruptive behavior

## Maintainer Responsibilities

Maintainers interpret and enforce this code of conduct.
When violations occur, maintainers may remove content, reject contributions, or apply further measures.

## Scope

This code of conduct applies to:

- repository discussions, issues, pull requests, and code reviews
- project-related communication channels

## Reporting Violations

Please report violations confidentially to the maintainers.
For sensitive cases, public escalation through Issues is not appropriate.

## Possible Actions

1. warning and clarification
2. temporary collaboration restrictions
3. exclusion from further contributions

## Attribution

This code of conduct follows Contributor Covenant principles and is adapted for this repository.
